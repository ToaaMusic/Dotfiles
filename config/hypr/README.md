# My Hyprland Configuration ~ `ver 0.53`

See [Hyprland's official documentation](https://wiki.hyprland.org/) for more.

## Shortcuts

See [keybinds.conf](https://github.com/ToaaMusic/dotfiles/blob/main/config/hypr/hyprmain/keybinds.conf)

`ALT` is referred to as `SUPER`.

### general keybinds

| Shortcut               | Action                          |
|------------------------|---------------------------------|
| `SUPER + C`            | Close active window             |
| `SUPER + M`            | Exit hyprland                   |
| `SUPER + V`            | floating mode for active window |


### specific keybinds

| Shortcut               | Action                          |
|------------------------|---------------------------------|
| `SUPER + Q`            | Open kitty                      |
